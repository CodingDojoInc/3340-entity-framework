RapperAPI

For this assignment, we'll be using the same JSON dataset, but now we'll add some routing to have this application function as an API, where our routes will return JSON data based on the route parameters provided.

Clone the repo below. The root route provides an outline of the expected routes:

  • '/artists' - a list of all artists
    • '/artists/Name/{string}' - a list of all artists whose names contain the string parameter
    • '/artists/RealName/{string}' - a list of all artists whose real names contain the string parameter
    • '/artists/Hometown/{string}' - a list of all artists whose hometown matches the string parameter
    • '/artists/GroupId/{int}' - a list of all artists who belong to the group associated with the given id
  • '/groups' - a list of all the groups
    • '/groups/Name/{string}' - a list of all groups whose names contain the string parameter
    • '/groups/GroupId/{int}' - the group whose id matches the specified value
    • (BONUS) ?ListArtists={boolean} - if ListArtists is true on a groups route, populate each displayed group with its list of artists
