Lecture 

LINQ Basics

Objective

Students will be able retrieve and filter data using LINQ.

Lesson Content 

  • Explain how LINQ methods operate on collection types (IEnumerable)
  • Explain how LINQ will work with the main ORM we will use (Entity Framework)
  • Cover how lambda expressions work with LINQ method calls
  • Go over some of the more useful LINQ methods for DB operations
    • .Where()
    • .Any()/.All()
    • .FirstOrDefault()
    • .Select()
    • .Min()/Max() => also how these can be overloaded
