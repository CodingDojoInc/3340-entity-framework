RESTful Routing

Learning Objectives:

  • Subscribe to RESTful Routing standards in an ASP.NET Core MVC application.

One thing we have been demonstrating throughout these modules but not overtly addressing is the use of RESTful routing in our code.

REST stands for Representational State Transfer. It is a set of standards used to create efficient, reusable routes for CRUD operations. Using this standard makes it easier for other developers to understand and navigate an application and results in clean, consistent URL paths for users.

As you have proceeded through this section, we have been demonstrating RESTful routing conventions, and it is in your best interest to know how to use them too.
