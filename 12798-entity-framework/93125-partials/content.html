Partials

Sometimes we want to render the same piece of HTML code over and over or have the same header across all of our pages. In situations like these, we can use partials: view files that represent a piece of a whole page.  The contents of a Partial view file isn't any different from a view, say, that is loaded from within a Layout.  That is, we can define just the markup and view logic (embedded C#) without the need to provide a full HTML document.  What separates Partials from full Views is how they are used, rather than how they are written.

Requirements For Use

To utilize Partials, we will need to enable Tag Helpers in a _ViewImports.cshtml file.  Make sure this file is present - in YourProject/Views/_ViewImports.cshtml - and has the following line of code:

_ViewImports.cshtml

// other imports
@addTagHelper *, Microsoft.AspNetCore.Mvc.TagHelpers

Creating a Partial

Let's say we want to create a simple navbar that we would like to re-use.  Simply create a new .cshtml file inside of your /Views directory.  If you would like to utilize the partial from any of your controllers, then place in /Views/Shared.  Otherwise, you could place the file from a particular View directory - say /Views/Home - if you only wish to load your Partial from a single controller. 

The same logic concerning View finding that applies to your normal Views also applies to Partials

In this example, let's place the partial in the same directory as the View from which it is being loaded.


NavPartial.cshtml

<nav>
    <ul>
        <li><a href="/">Home</a></li>
        <li><a href="/about">About Me</a></li>
    </ul>
</nav>

Loading a Partial

To load a partial, simply just use the partial tag helper.  This will look like any old HTML element, but it is unique to ASP Razor, and requires that you have tag helpers available to your Views (see above!).  This element takes a name attribute that will refer to the name (sans .cshtml file extension) of the Partial you wish to load.

Index.cshtml

<partial name="NavPartial"/>
<h1>Welcome to this site!</h1>
<p>The rest of your View content here...</p>
