Bank Accounts

Learning Objectives:

  • Construct a One-to-Many relationship between two models using Entity Framework Core.
  • Apply logic to prevent unauthorized functionality such as over-withdrawing and attempting to access other users' account pages.

Directions

Create an app that allows users to register and log in. After a user logs in, they should be taken to an account page that displays their bank records.

  • All validations from the Login and Registration assignment apply here
  • Users should be shown an error if they try to withdraw more than their balance
  • Display the user's name at the top of the page
  • If a user tries to access another user's account page, they should be redirected to their own account
  • Withdraw and deposit use the same form, which should accept positive or negative numbers
  • Bonus: Change the balance text color from green to red if the account total reaches $0
